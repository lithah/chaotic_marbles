**Breakout like game made to learn how to LÖVE <3**

I am trying to understand and learn how to code with love2d; and this project shows the progress made with it!

- Menu
- Level Creator! (beta functionality; but works for the task)
- 13 Levels (some done, but all playable!)
- Powerups! (Enlarge, Laser, Melt, Extra life)
- Settings
- SFX & Music (by TeknoAxe, check it out!)
